静态缓存实践

2016-12-12 PHP

静态缓存

什么是静态缓存？简单理解就是保存在磁盘上的静态文件，将生成的数据放入静态文件中，一般是存储不经常变动内容的页面数据。

思路：

  • 创建缓存文件

命名方式比如：以key作为文件名。先指定一个文件夹比如files用于缓存文件的文件夹，然后根据key和path创建具体缓存文件

  • 封装在一个方法中，根据参数做不同的操作（当然也可以封装三个方法）

cacheData($key, $value = '', $path = '')

//get cache
$value === ''

//delete cache
$value !== '' && is_null($value)

//set cache
$value !== '' && !is_null($value)

用到的php文件处理函数

//获取缓存文件的文件（上级）目录
$dir = dirname($filename); 

//判断文件夹(目录)是否存在
is_dir($dir); 

//创建文件目录
mkdir($dir, 0777);

//判断文件是否存在
is_file($filename);

//删除文件
unlink($filename);

封装静态缓存操作类

class Scache{
    const EXT = '.txt';
    private $_dir;
    public function __construct(){
        $this->_dir = dirname() . '/files/';
    }

    //一个函数，不同参数，产生【写缓存，读缓存，删缓存】
    public function cacheMeta($key, $value = '', $path = ''){
        $filename = $this->_dir . $path . $key . self::EXT;
        if ($value !== ''){
            if (is_null($value)){
                return @unlink($filename);
            }

            $dir = dirname($filename);
            if (!is_dir($dir)){ //判断目录是否存在
                mkdir($dir, 0777);
            }

            //写入成功返回字节数，失败返回false
            return file_put_contents($filename, json_encode($value)); 
        }

        if(!is_file($filename)){
            return false;
        }
        return file_get_contents($filename);
    }

}

操作

include_once 'scache.php';

$obj = new Scache;

//设置缓存
$key = 'learn_upeng_001';
$value = array(
    'a' => 'learn file cache function',
    'b' => array(
        'c' => 'for learn filename operation',
        'd' => 'learn oop at the same time',
    )
);
$obj->cacheMeta($key, $value);

//读取缓存
$obj->cacheMeta($key);

//删除缓存
$obj->cacheMeta($key, NULL);
